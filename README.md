# Simulador de la Tercera Ley de Newton

Un simulador interactivo desarrollado con HTML5 Canvas para visualizar los principios de Acción y Reacción. Este proyecto permite experimentar con diferentes masas y fuerzas de impacto para observar cómo se comportan los objetos según las leyes de la física clásica[cite: 3].

## Características Principales
* Cálculo en tiempo real de Aceleración (a = F/m)[cite: 3].
* Visualización dinámica de vectores de fuerza (Acción y Reacción)[cite: 3].
* Sistema de cronometraje preciso para el análisis del impacto[cite: 3].
* Interfaz responsiva con diseño oscuro y acentos neón[cite: 3].
* Control total sobre variables de entrada (Masa A, Masa B y Fuerza)[cite: 3].

## Tecnologías Utilizadas
* **HTML5**: Estructura semántica y elemento Canvas para el renderizado gráfico[cite: 3].
* **CSS3**: Diseño basado en Flexbox, tipografía personalizada y efectos de iluminación neón[cite: 3].
* **JavaScript (Vanilla)**: Lógica física, motores de animación y manejo del DOM[cite: 3].

## Fórmulas Aplicadas
El simulador se basa en la relación fundamental entre la fuerza, la masa y la aceleración:
F = m * a  -->  a = F / m[cite: 3]

Según la Tercera Ley, para cada fuerza de acción existe una fuerza de reacción de igual magnitud y dirección opuesta:
F1 = -F2[cite: 3]
